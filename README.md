## How to play
| Keys | Action |
| ----- | ----- |
| A D | Move player |
| Space | Jump |
| W | Climb ladder |
| 1-9 | Switch inventory hot bar |
| Enter | Give blocks |

## Dev Commands

Run dev mode (auto-update code when editting)
```sh
npm run dev
```

Run build
```sh
npm run build
```
